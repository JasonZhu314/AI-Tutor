# Privacy and Data Boundaries

This project is intended to be public-safe. Real learner memory should stay private.

## Public Repo May Contain

- source code;
- generic templates;
- synthetic examples;
- command specs;
- design docs;
- public references.

## Public Repo Must Not Contain

- real learner profiles;
- real course notes;
- real session logs;
- actual misconceptions;
- private Obsidian vault pages;
- copyrighted course materials;
- PDFs, slides, transcripts, or textbooks;
- API keys or tokens.

## Recommended Private Locations

Keep real AI Tutor state under your Obsidian vault, for example:

```text
Obsidian/Projects/AI Tutor/
  Global Learner Profile.md
  Domains/
  Workspaces/
  Session Logs/
```

If using local workspace control files, keep private state ignored by git.

## Synthetic Examples

Examples in this repository should be synthetic or use public toy materials.
