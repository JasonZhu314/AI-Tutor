# nanoGPT Example

This is a synthetic example placeholder for the first codebase pilot.

Do not copy private learner state into this folder. Public examples should use generic goals and public source references.

## Example Goal

Learn the nanoGPT codebase as a path into GPT internals.

1. Repo map and learning goals
2. Character-level language modeling
3. Tokenization: character-level vs GPT-2 BPE
4. Forward pass: `idx -> embeddings -> blocks -> logits`
5. Causal self-attention tensor shapes
6. Next-token cross entropy
7. Training loop
8. Sampling with temperature and top-k
9. Scaling notebooks
10. Extension project
