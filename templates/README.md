# Templates

These templates are public-safe starting points. Real learner state should be instantiated in a private Obsidian vault or private workspace, not committed to this repository.

## Template Groups

```text
global/      Cross-subject learner memory
domain/      Domain-specific learner state
workspace/   Per-folder workspace controls
```

## Usage

The future CLI will copy these templates into configured locations and replace placeholders such as:

```text
{{learner_name}}
{{domain}}
{{workspace_name}}
{{source_root}}
{{created_date}}
```
